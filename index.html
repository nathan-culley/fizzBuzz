FizzBuzz

This is my attempt at creating FizzBuzz for The Odin Project. The Odin Project only requires that this be a console program, but I was getting rusty on my HTML and CSS at the time of programming this, so I decided to make a full-fledged webpage instead.

FizzBuzz is a classic programming puzzle for beginners. The goal is to write a program that outputs a sequence of numbers, starting at 1 and ending at a number chosen by a user. The catch is that numbers that are divisible by 3 are replaced in the sequence with "Fizz," numbers that are divisible by 5 are replaced with "Buzz," and numbers that are divisible by both 3 and 5 are replaced with "FizzBuzz."

For example, if the user chose the number 19, the sequence would be "1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz 11 Fizz 13 14 FizzBuzz 16 17 Fizz 19." Try it yourself below:
